# プライバシーポリシー - TORU JAN

「TORU JAN」（以下、「本アプリ」といいます）は、ユーザーのプライバシーを尊重し、個人情報の保護に努めます。

### 1. 収集する情報とその利用目的
本アプリは、以下の目的のために端末の機能を使用しますが、個人を特定できる情報を収集・サーバーへ送信することはありません。

* **カメラへのアクセス**: JANコードをスキャンし、バーコードを生成するために使用します。画像データはデバイス内での解析（Google ML Kitを使用）にのみ利用され、外部に送信されることはありません。
* **バイブレーションの制御**: 読み取り完了時のフィードバック（通知）のために使用します。

### 2. 第三者提供について
本アプリは、収集した情報を第三者に提供することはありません。

### 3. 免責事項
本アプリの利用により生じた損害、トラブル等について、開発者は一切の責任を負いません。読み取った内容の正確性は、必ずユーザー自身で確認してください。

### 4. プライバシーポリシーの変更
本ポリシーの内容は、ユーザーに通知することなく変更することができるものとします。
