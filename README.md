# TORU JAN (トルジャン) 📸

AndroidカメラでJANコード（数字）を高速スキャンし、店舗業務を効率化するツールです。
[企画書はこちら](./TORU%20JAN%20企画書.pdf)

## 📝 プロジェクト概要
ダイソーなどの店舗における「返品処理」の負担を軽減するために開発しました  
複雑なJANコードの手入力作業を、カメラによる自動認識と直感的な操作感で解決します。

### 💡 開発の背景と目的
- **課題**: セルフレジ導入店舗等において、返品時に13〜14桁の長いJANコードを全て手入力する必要があり、1点あたり30秒〜1分以上の時間がかかっていました。
- **解決策**: カメラによるリアルタイム数字認識を導入し、読み取ったデータから瞬時にバーコードを再生成。手入力の手間とミスをゼロにします。
- **ターゲット**: 店舗従業員、特にJANコードの入力が苦手な方やPC操作に不慣れな方。

## 🚀 主な機能
- **リアルタイムJAN認識**: Google ML Kit を使用し、カメラをかざすだけで数字を抽出。
- **確定フィードバック**: 
  - **聴覚**: ビープ音による通知。
  - **触覚**: バイブレーションによる「カチッ」とした操作感。
- **バーコード生成**: 認識した数字からバーコードを画面に表示し、レジでのスキャンを可能にします。
- **履歴管理**: 読み取った内容をセッションごとに保存し、後から確認・再利用が可能。
## 使い方の例  
**[返品処理]**  
  
 **1.バーコードのアイコンをタップ。**  
 **2.カメラのアイコンをタップ。**  
 **3.緑の枠内にレジに表示されたJANコードを収めます。**  
 **4.読み取った数字が正しければ、OKボタンを押します。**  
 **5.表示されたバーコードをレジでスキャンします。**  
 **6.→ボタンを押して2~5を繰り返します。**  
 **7.すべての商品をスキャンし終えたら、家ボタンを押します。**  

**[返品処理後のの再精算]**  
  
返品処理時のデータを再利用できます。 
  
**1.時計マークをタップ。**  
**2.先ほどの商品のリストが表示されます。最新のは一番上の項目なので、タップ。**  
**3.一番上のJANコードをタップ**  
**4.あとは矢印で次々とバーコードが表示されるのでレジで読み取ってください。**  

**[商品検索]**  
  
お客様が商品のJANコードを表示してくれた場合に、カメラで数字を読み取ってバーコードを生成し、店舗のPCのバーコードリーダーで読み取って検索できます。手書きの数字でも認識します!
  
**バーコードの印字不良**  
  
商品や値下げシールの印字不良でバーコードがかすれていたり、にじんでいる場合、JANコードを読み取ってバーコードを作れます。

## ✨ こだわりポイント
- **現場に即したUI/UX**: PCが苦手な方でも迷わないよう、文字説明を最小限にし、大きく理解しやすいアイコンを採用しています 。
- **誤認識防止ロジック**: `REQUIRED_STABILITY` パラメータによる連続一致判定を実装し、精度の高い読み取りを実現しています。
- **実用的なカメラ設定**: 精度向上と枠への収めやすさを考慮し、ズーム倍率を 1.7 倍に固定しました 。
- **確実な操作フロー**: 誤ったJANで画面遷移することを防ぐため、自動遷移ではなく「確定ボタン」による手動遷移を採用しています 。

## 🛠 使用技術
- **Language**: Kotlin
- **Camera**: CameraX (Jetpack)
- **AI/ML**: Google ML Kit (Text Recognition)
- **Sound/Haptics**: ToneGenerator, MediaPlayer, VibratorManager
- **UI**: Material Components / ViewBinding
