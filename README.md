# TORU JAN (トルジャン) 📸

AndroidカメラでJANコード（数字）を高速スキャンし、店舗業務を効率化するツールです。
[企画書はこちら](./TORU%20JAN%20企画書.pdf)

## 📝 プロジェクト概要
[cite_start]ダイソーなどの店舗における「返品処理」の負担を軽減するために開発しました [cite: 4, 5]。
複雑なJANコードの手入力作業を、カメラによる自動認識と直感的な操作感で解決します。

### 💡 開発の背景と目的
- [cite_start]**課題**: セルフレジ導入店舗等において、返品時に13〜14桁の長いJANコードを全て手入力する必要があり、1点あたり30秒〜1分以上の時間がかかっていました [cite: 4]。
- [cite_start]**解決策**: カメラによるリアルタイム数字認識を導入し、読み取ったデータから瞬時にバーコードを再生成。手入力の手間とミスをゼロにします [cite: 5, 12]。
- [cite_start]**ターゲット**: 店舗従業員、特にJANコードの入力が苦手な方やPC操作に不慣れな方 [cite: 7]。

## 🚀 主な機能
- [cite_start]**リアルタイムJAN認識**: Google ML Kit を使用し、カメラをかざすだけで数字を抽出 [cite: 12]。
- **確定フィードバック**: 
  - **聴覚**: ビープ音による通知。
  - **触覚**: バイブレーションによる「カチッ」とした操作感。
- [cite_start]**バーコード生成**: 認識した数字からバーコードを画面に表示し、レジでのスキャンを可能にします [cite: 12]。
- [cite_start]**履歴管理**: 読み取った内容をセッションごとに保存し、後から確認・再利用が可能 [cite: 12]。

## ✨ こだわりポイント
- [cite_start]**現場に即したUI/UX**: PCが苦手な方でも迷わないよう、文字説明を最小限にし、大きく理解しやすいアイコンを採用しています 。
- **誤認識防止ロジック**: `REQUIRED_STABILITY` パラメータによる連続一致判定を実装し、精度の高い読み取りを実現しています。
- [cite_start]**実用的なカメラ設定**: 精度向上と枠への収めやすさを考慮し、ズーム倍率を 1.7 倍に固定しました 。
- [cite_start]**確実な操作フロー**: 誤ったJANで画面遷移することを防ぐため、自動遷移ではなく「確定ボタン」による手動遷移を採用しています 。

## 🛠 使用技術
- **Language**: Kotlin
- **Camera**: CameraX (Jetpack)
- **AI/ML**: Google ML Kit (Text Recognition)
- **Sound/Haptics**: ToneGenerator, MediaPlayer, VibratorManager
- **UI**: Material Components / ViewBinding
